*************************
Introduction
*************************
Dans le cadre du projet R&D, nous avons choisi de mettre en œuvre une solution logicielle de lecture de flux RSS pour le système d’exploitation mobile Firefox OS.
Nous sommes partis pour ce projet de recherche et développement sur l’idée de proposer un moteur de récupération et traitement de flux RSS.
Ce projet a pour exigence de respecter scrupuleusement l’utilisation de technologies web libres et de garder en ligne de mire une utilisation off-line.
De plus, FireFox OS a fait naitre en nous un intérêt assez fort et la motivation de développer une application dédiée pour cette plateforme.
